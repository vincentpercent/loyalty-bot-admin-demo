# Media placeholders

Оригинальные медиафайлы убраны из демо-репозитория. Для работы сценариев можно добавить свои GIF/MP4/изображения в соответствующие пути (`bot/media/gifs`, `bot/media/cats`). Если файлов нет, бот gracefully упадёт на текстовые ответы благодаря проверке наличия файлов в `send_media_with_fallback`.
